Features

Open Source

Bootstrap is an open source toolkit for developing with HTML, CSS, and JS. Quickly prototype your ideas or build your entire app with our Sass variables and mixins, responsive grid system, extensive prebuilt components, and powerful plugins built on jQuery.

Features

Mobile First

Bootstrap includes a powerful mobile-first flexbox grid to build layouts of all shapes and sizes thanks to a twelve column system, five default responsive tiers, Sass variables and mixins, and dozens of predefined classes.

Features

Components Library

Bootstrap comes with an extensive components library that includes buttons and lists to carousels and modals. Components are easy to integrate into any site and require no knowledge of JavaScript to use.

How to use Bootstrap

Bootstrap uses a mobile-first 12 column flexbox grid to build layouts. The first step is to create a container that all the content will reside in by using class="container". The second step is to define your rows within the container with class="rows". The third step is too define the size of each column within that row by using class="col-{breakpoint}-{# of columns}".

Example:

col-12
col-6
col-6
col-4
col-4
col-4
col-12

Now that you have your grid set up you can now add classes to your html tags to create and style components.

Components

Buttons

The button component allows you to create buttons quickly with predefined styles,sizes and states.

Cards

The card component is a content container that allows for various customization. Such as headers, titles, body text, footers, images, buttons, lists and more.
